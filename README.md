This is Arithmetic Trainer, a console app for practising basic arithmetic.

Pick a practice mode:

```txt
Welcome to Arithmetic Trainer
Pick practice mode:  
                     
> Addition (+)       
  Subtraction (-)    
  Multiplication (*) 
  Division (/)       
  All Together       
  Cancel
```
